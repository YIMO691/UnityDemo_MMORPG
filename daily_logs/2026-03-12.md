# 二、完成内容

## 1. UI 框架重构与遮罩系统

- BasePanel：统一生命周期（OnCreate/OnShow/OnHide/OnDestroyPanel），内置淡入淡出与交互控制，提供 HideImmediately。
- UIManager：  
  - 初始化 PanelCanvas 与层级映射（Bottom/Normal/Popup/Top）。  
  - 主页面 ShowMainPage 支持隐藏旧页与复用。  
  - Popup 栈管理（TryPushPopup/RemovePopupFromStack/RefreshPopupMask），与遮罩联动。  
  - 事件接入（OpenPanelEvent、ClosePanelEvent、OpenMainPageEvent、OpenRoleInfoPanelEvent）。
- UIMask：控制遮罩显示/隐藏与点击关闭顶层 Popup（当 CloseByMask 为真）。

## 2. BeginPanel 与相机转场

- 开始游戏：播放 CameraEvent.TurnAround 动画，动画结束回调切换至 CreateRolePanel。  
- 返回逻辑：从创角页返回主菜单时复位按钮状态并重置相机。  
- 设置/关于/退出：设置面板弹窗可用，关于保留后续接入，退出兼容编辑器与运行时。

## 3. 创建角色系统（UI + 事件 + 数据）

- CreateRolePanel（Normal）：  
  - 左右切换职业、展示名称与头像；输入昵称；查看详情；提交创角请求。  
  - 发布 OpenRoleInfoPanelEvent 展示详情；发布 CreateRoleRequestEvent 提交请求。
- RoleInfoPanel（Popup）：  
  - 遮罩可点击关闭；展示职业定位、描述与基础属性。
- 事件与流程：  
  - CreateRoleFlowController 订阅 CreateRoleRequestEvent，调用 PlayerFactory 生成 PlayerData 并写入 DataManager。
- 配置与资源：  
  - RoleDataManager 从 Resources/Config/RoleClassConfig.json 加载职业配置；Portrait 下新增 4 张头像。

## 4. 设置与音频联动

- SettingPanel：音乐/音效开关与音量滑条双向绑定；即时生效（AudioManager）并持久化（DataManager + JsonMgr）。  
- AudioManager：开机 ApplySetting；缺省音源与组件自创建，避免空引用。

## 5. 事件系统与启动流程

- EventBus：统一发布/订阅/清理，空订阅安全返回。  
- StartGame：启动时初始化 UIManager 与 CreateRoleFlowController，展示 BeginPanel 作为主页。

# 三、影响范围与兼容性

- 依赖 PanelCanvas 的层节点命名需与 UILayer 枚举一致，否则遮罩与层级挂载异常。  
- CameraEvent 需配置 Animator 与 Turn 触发器，且在动画末尾调用 PlayerOver 事件；缺失时将直接跳过动画进入创角页。  
- 资源路径基于 Resources，若未来切换至 AssetBundle/Addressables，需扩展 ResourceManager 的加载实现。
