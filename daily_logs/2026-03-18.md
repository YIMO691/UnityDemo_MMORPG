# 2026-03-18 每日变更日志

## 新功能
- 地图系统（Map System）
  - MapPanel：显示地图名称、地图图片；玩家标点实时映射（X-Z 平面）。
  - MapConfig：支持每张地图的世界边界 worldMinX/worldMaxX/worldMinZ/worldMaxZ。
  - MapDataManager：从 Resources/Config/MapConfig.json 加载配置。
  - MapService：获取当前场景与玩家位置（含 activeScene 兜底），提供当前地图配置。
  - PlayerLocator：注册/获取玩家 Transform，优先实时位置。

- 自动寻路系统（NavigationSystem）
  - Contracts：INavigationAgent、NavigationMoveRequest。
  - Events：NavigationMoveRequestEvent、NavigationStopRequestEvent。
  - Runtime：NavigationPathSolver（NavMesh.SamplePosition/CalculatePath）、NavigationRegistry、NavigationService（订阅/分发）。
  - Components：BaseNavigator（通用路径驱动）、PlayerNavigator（基于 StarterAssetsInputs 的玩家移动）。
  - UI：MapClickReceiver（点击地图 → 世界坐标 → 发布 NavigationMoveRequest）。

- 玩家 UI 与数据
  - MainPanel：HP=runtime.currentHp/attribute.maxHp，Stamina=runtime.currentMp/attribute.maxMp；头像由 RoleClassConfig.mainHeadId 动态加载（UIPaths.PortraitRoleHeadRoot）。
  - 角色创建：初始化 runtime.currentHp/currentMp 为 max 值。

## 架构与框架调整
- 数据归口
  - PlayerData/PlayerBaseData/PlayerAttributeData/PlayerProgressData/PlayerRuntimeData 迁至 Game/Entity/Player/Data 下（保 GUID）。
  - DataManager 收敛为设置/槽位与文件名基础 IO；玩家业务态由 GamePlayerDataService 承载。

- 初始化点
  - AppRuntimeInitializer：新增 MapDataManager.Init()、NavigationService.Init()。
  - GameSceneEntry：注册 PlayerLocator；写入 runtime.Scene 与 pos；挂 PlayerNavigator 并设 AgentId="Player"。

- 路径常量
  - UIPaths.PortraitRoleHeadRoot = "Portrait/RoleHead/"
  - UIPaths.MapImageRoot = "Map/Main/"
  - AssetPaths.MapConfig = "Config/MapConfig"

## 行为修复与体验
- 光标策略
  - StarterAssetsInputs：焦点返回时解锁并显示鼠标（Cursor.lockState=None，Cursor.visible=true），防止 UI 无法点击。

- 自动寻路取消判定
  - 增加 cancelGraceTime（0.8s），防止因 UI 操作抖动误取消。
  - 移除鼠标 delta/左键作为取消条件；仅保留 WASD/Space/右键/手柄摇杆。
  - 方向输入统一归一化以避免 analogMovement 降速。
  - 增加日志：SetPath 角点数与取消原因打印。
