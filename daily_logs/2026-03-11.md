## 4. UIFramework 基础搭建

当前 UI 框架已实现：

-   UIManager
-   BasePanel
-   UILayer 分层
-   Popup 遮罩系统
-   Panel 缓存

UI 层级结构：

Canvas ├ Bottom ├ Normal ├ Popup └ Top

------------------------------------------------------------------------

## 5. Panel 生命周期统一

统一 Panel 生命周期结构：

-   OnCreate：首次创建
-   OnShow：每次显示
-   OnHide：每次隐藏
-   OnDestroyPanel：销毁前清理

通过 `BasePanel` 统一管理 UI 状态。

------------------------------------------------------------------------

## 6. Popup UI 管理

实现 Popup 弹窗基础管理：

-   PopupStack 栈结构
-   UIMask 遮罩系统

-   点击遮罩关闭 Popup
